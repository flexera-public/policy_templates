# Google Cloud Data

Various Google-specific data assets, such as pricing data. Some assets are generated via scripts in `tools/cloud_data/google/` (run manually as needed); see the [Cloud Data Scripts README](https://github.com/flexera-public/policy_templates/blob/master/tools/cloud_data/README.md) for more information. Smaller reference files (`instance_types.json`, `regions.json`) are manually maintained.
